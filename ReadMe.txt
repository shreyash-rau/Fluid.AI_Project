# PDF Keyword Extractor

This Python script extracts specific keywords from a PDF file. It's designed to help investors evaluate a company by extracting key information from a company's reports or documents.

## Dependencies

- PyPDF2: A Python library used for reading PDF files.
- nltk: The Natural Language Toolkit, a Python library used for working with human language data.

You can install these libraries using pip:

```bash
pip install PyPDF2 nltk
